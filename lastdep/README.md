# LastDep
2D кооперативная аркадная игра с элементами RPG и видом сверху. Игроки перемещаются по ограниченной карте в виде ярмарки. Для двух игроков предусмотрены взаимодействия и парные мини-игры расположенные на карте, в которых вы будете соперничать со вторым игроком. Главная цель — выиграть своего противника в большем количестве мини-игр и получить за это секретный приз.
